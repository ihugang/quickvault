# QuickVault 优雅 UI 设计

## 🎨 设计理念

**核心价值**：知性、优雅、现代

- **知性**：智能交互，清晰的信息层级，引导式操作
- **优雅**：精致的视觉细节，流畅的动画过渡
- **现代**：符合 iOS 设计规范，使用最新 SwiftUI 特性

## 📱 视图架构

### 1. ItemListView - 主列表视图

**设计亮点**：
```
顶部搜索栏（圆角、毛玻璃效果）
    ↓
置顶卡片区（橙色 pin 图标）
    ↓
普通卡片区（时间倒序）
    ↓
空状态引导（优雅的图标和文案）
```

**视觉特性**：
- 🎨 渐变背景（微妙的层次感）
- 🔍 现代搜索栏（圆角 16pt，灰色背景）
- 📋 卡片设计（圆角 20pt，轻微阴影）
- 🏷️ 类型图标（蓝色=文本，紫色=图片）
- 📌 置顶标记（橙色 pin.fill）

**交互细节**：
- 点击卡片 → 进入详情页
- 右上角 + 按钮 → 菜单选择类型
- 搜索栏实时过滤
- 下拉刷新数据

### 2. ItemDetailView - 详情视图

**设计布局**：
```
顶部大图标（圆形，80x80）
    ↓
标题（粗体，居中）
    ↓
时间信息（小字，灰色）
    ↓
内容区域（白色卡片）
    ↓
标签云（流式布局）
    ↓
操作按钮（分享）
```

**特色功能**：

**文本卡片**：
- ✅ 内容可选择复制
- ✅ 一键分享文本
- ✅ 简洁的排版

**图片卡片**：
- ✅ 2列网格布局
- ✅ 点击放大查看
- ✅ 可缩放图片查看器
- ✅ 水印配置面板

**水印配置**：
```swift
Sheet 面板
├── 开关：是否添加水印
├── 输入框：水印文字
└── 预览：右下角显示位置
```

### 3. CreateItemSheet - 创建视图

**分步引导**：

**步骤 1：选择类型**
```
┌─────────────────────────┐
│  📄 文本卡片             │
│  存储文本信息，便于快速分享│  →
└─────────────────────────┘

┌─────────────────────────┐
│  🖼️ 图片卡片             │
│  保存图片，支持添加水印分享│  →
└─────────────────────────┘
```

**步骤 2：填写内容**
```
标题输入
    ↓
内容区域（文本编辑器 / 图片选择器）
    ↓
标签管理（流式布局，可添加删除）
    ↓
创建按钮（底部固定）
```

**图片选择器**：
- 使用 PhotosPicker（iOS 16+）
- 支持多选（最多 10 张）
- 实时预览缩略图
- 可单独删除

## 🎭 视觉语言

### 配色方案

```swift
文本类型：
- 主色：蓝色 (.blue)
- 浅背景：蓝色 10% 透明度

图片类型：
- 主色：紫色 (.purple)
- 浅背景：紫色 10% 透明度

通用：
- 背景：系统背景色
- 文字：系统主色/次要色
- 阴影：黑色 5% 透明度
```

### 圆角规范

```swift
卡片：20pt (continuous)
按钮：16pt (continuous)
图标容器：12-14pt (continuous)
标签胶囊：全圆角 (Capsule)
```

### 间距系统

```swift
超小：4pt  - 图标与文字
小：  8pt  - 标签间距
中：  12pt - 卡片内元素
大：  16pt - section 间距
超大：20pt - 页面边距
巨大：24pt - 顶部区域
```

### 字体层级

```swift
.title2.weight(.bold)    - 页面标题
.headline                - 卡片标题
.subheadline.weight(.semibold) - section 标题
.body                    - 正文内容
.subheadline             - 次要信息
.caption                 - 辅助信息
```
